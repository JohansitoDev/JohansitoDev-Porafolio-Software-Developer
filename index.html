Galería de proyectos generales tecnológicos

Explora mis soluciones de software, arquitectura de sistemas y análisis defensivo:

Ciberseguridad / Blue Team

APK Vulnerability & Malware Lab

¿Qué hace?: Realiza descompilación, ingeniería inversa y análisis estático estructurado sobre binarios y paquetes APK de Android.

¿Qué resuelve?: Automatiza la detección de credenciales quemadas en el código, debilidades críticas del estándar OWASP Mobile e inyección de código malicioso Smali antes del despliegue.

Finalidad: Proveer un ecosistema controlado y preventivo para auditar el software móvil corporativo, protegiendo al usuario de fugas de información.

Backend

Secure Auth & Log API Engine

¿Qué hace?: Gestiona el control de accesos mediante tokens cifrados avanzados (JWT), sanitización estricta de entradas y logs inmutables.

¿Qué resuelve?: Detiene de forma preventiva intentos comunes de inyección de código (SQLi, NoSQLi) y ataques continuos de fuerza bruta.

Finalidad: Servir como base estructural inquebrantable para soportar arquitecturas de datos y microservicios externos de manera confiable.

Frontend Design

UI Custom Analytical Dashboard

¿Qué hace?: Renderiza flujos complejos de datos informativos en una interfaz minimalista, fluida y altamente responsiva empleando Tailwind CSS.

¿Qué resuelve?: Elimina la fatiga visual del usuario y la carga innecesaria del DOM web al optimizar los recursos y componentes estáticos organizados.

Finalidad: Garantizar una experiencia de navegación limpia e intuitiva, acortando la curva de aprendizaje de cualquier software empresarial.

Mobile

Reporta RD Engine Core

¿Qué hace?: Procesa reportes e incidencias de infraestructura urbana mapeando coordenadas de geolocalización precisas desde el celular.

¿Qué resuelve?: Agiliza de forma drástica los canales burocráticos y lentos de reportes comunitarios tradicionales unificando los flujos.

Finalidad: Conectar de manera directa las problemáticas de la ciudadanía con respuestas tecnológicas rápidas de un MVP eficiente.

Videojuego Web

Canvas Retro RPG Core Engine

¿Qué hace?: Controla colisiones por celdas matemáticamente exactas, procesamiento de estados y dibujo nativo bidimensional a través del Canvas de HTML5.

¿Qué resuelve?: Elimina la dependencia estricta de motores de software pesados de terceros o librerías que inflan el ancho de banda del navegador web.

Finalidad: Estudiar e implementar algoritmos eficientes de renderizado gráfico de videojuegos clásicos optimizando la memoria RAM local.

AR Web Experimento

A-Frame Spatial AR Interactive Viewer

¿Qué hace?: Renderiza objetos tridimensionales interactivos superpuestos en el espacio físico real usando la cámara del teléfono mediante frameworks WebXR.

¿Qué resuelve?: Evita la necesidad obligatoria de forzar al usuario a descargar aplicaciones nativas pesadas en tiendas externas para vivir experiencias AR.

Finalidad: Explorar los límites actuales de la navegación web inmersiva de cara a soluciones interactivas futuras en navegadores modernos.
